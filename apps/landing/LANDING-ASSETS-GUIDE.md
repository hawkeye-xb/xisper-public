# Xisper Landing Page 图片素材准备指南

## 页面结构概览

```
Hero（首屏）
  └─ App Icon ✅ 已有
Highlights（亮点条）
  └─ 纯文字 + 图标，无需图片
RoleShowcase（角色词库展示）
  └─ 纯文字交互，无需图片
CoreFeatures（四大核心功能）
  ├─ 角色热词 ← 需要截图
  ├─ 语音听写 ← 需要截图
  ├─ AI 润色  ← 需要截图
  └─ 全局可用 ✅ 代码实现的 App Icon 墙，无需图片
HowItWorks（三步使用）
  └─ 纯文字 + 图标，无需图片
FeatureGrid（更多特性）
  └─ 纯文字卡片，无需图片
FinalCTA（底部下载）
  └─ 纯文字，无需图片
```

## 需要准备的图片：3 张

所有图片放到 `apps/landing/public/` 目录下。

### 1. `feature-roles.png` — 角色热词

**对应功能**：「你的职业，你的词库」

**建议内容**：
- Xisper 设置界面中角色/热词切换的截图
- 或者一个示意图：左侧是角色列表（开发者/律师/医生），右侧是对应的热词标签云
- 重点体现「切换角色 → 词库跟着变」的概念

**尺寸**：800×600px 或更大，PNG，深色背景（与页面协调）

---

### 2. `feature-dictation.png` — 语音听写

**对应功能**：「自然说话，精准转写」

**建议内容**：
- 屏幕录制截图：某个应用中光标处正在出现转写文字
- 或者包含录音气泡（LiveTranscribePanel）的截图
- 重点体现「在任意应用中，按 Fn 说话，文字实时出现」

**尺寸**：800×600px，PNG

---

### 3. `feature-polish.png` — AI 润色

**对应功能**：「口语变书面，一步到位」

**建议内容**：
- 左右或上下对比图：原始口语转写 vs AI 润色后的文字
- 用高亮/删除线标注 AI 修改的部分
- 或者截图 Xisper 历史记录界面（HistoryView），展示 raw vs processed 文字

**尺寸**：800×600px，PNG

## 图片风格要求

| 项目 | 要求 |
|------|------|
| 背景色 | 深色（#111820 附近）或透明，与页面深色主题融合 |
| 圆角 | 图片内容如有窗口截图，保持 macOS 原生圆角 |
| 分辨率 | 至少 800×600px，推荐 1600×1200px（Retina） |
| 格式 | PNG（透明背景优先）或 JPEG |
| 文字 | 图中如有文字内容，使用中文 |
| 品牌色 | 高亮/强调色使用天青色 `#007C8B` |
