1. 项目概述
本架构旨在提供一套基于 Cloudflare 生态 的 Serverless AI 应用标准化方案。支持多租户、自动限流、身份验证及文件生命周期管理。

2. 技术栈 (The Golden Stack)
Gateway/Logic: Hono (Lightweight Web Framework)

Auth: Logto (OIDC, Google/Alipay SSO)

Database: Cloudflare D1 (SQLite on Edge)

Cache/Quota: Cloudflare KV (Low latency key-value)

Storage: Cloudflare R2 (S3-compatible, Zero egress fee)

3. 数据模型设计
3.1 用户扩展表 (D1: users)
用于存储 Logto 以外的业务数据。

SQL
CREATE TABLE users (
  id TEXT PRIMARY KEY,       -- 对应 Logto 的 sub (Subject ID)
  email TEXT,
  tier TEXT DEFAULT 'free',  -- free, pro, vip
  created_at INTEGER,        -- Unix timestamp
  metadata TEXT              -- JSON 扩展字段
);
3.2 任务记录表 (D1: tasks)
SQL
CREATE TABLE tasks (
  id TEXT PRIMARY KEY,
  user_id TEXT,
  status TEXT,               -- pending, processing, completed, failed
  r2_key TEXT,               -- 指向 R2 的路径
  tokens_used INTEGER,
  created_at INTEGER
);
4. 核心逻辑 SOP
4.1 身份验证流程
前端集成 Logto SDK（Web 使用 @logto/react 或 @logto/browser，移动端使用对应平台 SDK）。

认证模式选择:
  - 方案 A（推荐快速开发）: 跳转到 Logto 托管登录页，支持自定义品牌色、Logo 和主题
  - 方案 B（完全自定义）: 纯 SDK 集成，自己实现登录 UI，直接调用 Logto API

前端获取 access_token 后，请求后端 API 时携带 Authorization: Bearer <token>。

Hono 中间件校验 JWT 合法性（验证签名、过期时间、issuer 等），并将 sub (user_id) 注入上下文。

首次登录时，后端自动在 D1 的 users 表中创建用户记录（或更新现有记录），实现 Logto 用户与业务数据的绑定。

多端支持:
  - Web 端: 创建 Single Page App 类型的 Logto Application
  - 移动端: 创建 Native App 类型的 Logto Application（需要时）
  - 后端: 无需单独的 Logto Application，只负责验证 token

4.2 文件上传与清理 (R2 Lifecycle)
临时存储：文件上传至 temp/{userId}/{uuid}。

清理规则：在 R2 控制台配置：

Prefix: temp/, Action: Delete after 1 day.

转正逻辑：AI 处理成功后，Worker 调用 R2.put() 将文件移至 final/{date}/{uuid}，并删除 temp 文件。

4.3 阶梯限流 (KV Quota)
读取：KV.get("quota:{userId}")。

校验：如果当前值 > 计划阈值，返回 429 Too Many Requests。

更新：API 调用成功后，KV.put 异步更新计数。

5. 快速部署指南 (SOP)
创建基础设施：

Bash
npx wrangler kv:namespace create MY_KV
npx wrangler d1 create MY_DB
npx wrangler r2 bucket create MY_BUCKET
配置密钥：

Bash
npx wrangler secret put LOGTO_ENDPOINT
npx wrangler secret put AI_API_KEY
数据库初始化：

Bash
npx wrangler d1 execute MY_DB --file=./schema.sql
一键发布：

Bash
npm run deploy
