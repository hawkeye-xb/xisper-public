# Xisper Design Files

This directory contains design files created with Pencil.dev for different parts of the Xisper project.

## Files

### `xisper-web-design.pen`
Design system for the web application (`apps/web`).

**Contents:**
- Analytics Screen: Statistics cards + 7-day chart
- History Screen: Date-grouped transcription history
- Settings Screen: Configuration panels
- Component Library: Buttons, badges, input fields
- Design Variables: 26 design tokens (colors, spacing, typography, radius)

**Documentation:** `apps/web/DESIGN_SYSTEM.md`

### `xisper-landing-design.pen`
Landing page design (created Feb 26, 2026).

---

## Opening Design Files

These files can be opened with:
1. **Pencil.dev Desktop App**: Download from https://pencil.dev
2. **Pencil MCP Tool**: Use in Cursor with Pencil MCP integration

## Design Tokens

All design tokens are synchronized with the codebase:
- `apps/web/src/styles/tokens/colors.css`
- `apps/web/src/styles/tokens/typography.css`
- `apps/web/src/styles/tokens/spacing.css`
- `apps/web/src/styles/tokens/effects.css`
