# 3.2 纠错包 — 你的口音，它都懂

> 核心场景：不是"高级功能展示"，而是解决一个真实的日常痛苦。

## 时长：60秒

## 脚本

| 时间 | 画面 | 旁白/字幕 |
|------|------|----------|
| 0-10秒 | 文字快速闪过，每行一个："dev → 戴夫？""merge → 默基？""staging → 死得金？""schema → 斯个马？""async → 阿辛克？" | "你说的是这些，但语音识别听到的是这些。" |
| 10-15秒 | Xisper 热词页面 → 开启纠错包的开关（清晰展示） | "一个开关搞定。" |
| 15-30秒 | 正常说一段中英混杂的话（用最自然的中式发音，不要刻意标准），展示所有术语精准识别 | "你就用你平时的发音说。dev、merge、staging、schema、async——全都对。" |
| 30-42秒 | 展示不同行业包的适用性：快速切一下法律包（"追诉时效"、"管辖权"）和金融包（"市盈率"、"杠杆"、"对冲"） | "不只是程序员。法律、医学、金融——每个行业都有自己容易听错的词。" |
| 42-52秒 | 展示自定义热词：添加一个自己公司的专有名词 | "还能加你自己的词。公司名、产品名、缩写——随你加。" |
| 52-60秒 | Logo + 标语 | "Xisper。口音不标准？没关系。" |

## 录制要点

- 0-10秒的文字闪过效果很重要，能让人瞬间产生共鸣（"这不就是我吗"）
- 说话时**不要故意用标准英语**，就用你平时和同事说话的发音
- 如果 ASR 识别结果太好了（不开纠错包也能认），换几个更难的词试试
