# 2.5 行业专业词汇 — 法律/医学/金融

> 目标用户：每天都要打大量行业术语的专业人士。

## 时长：90秒

## 脚本

| 时间 | 画面 | 旁白/字幕 |
|------|------|----------|
| 0-8秒 | 标题卡："每个行业，都有自己的语言。" | |
| 8-20秒 | 打开 Xisper → 热词页面 → 展示纠错包区域，6 个包：开发者、法律、医学、金融、通用商务、学术 | "6 个行业纠错包，开箱即用。每一个都针对你的领域微调识别精度。" |
| 20-22秒 | **法律篇开始** — 打开"法律"纠错包 | |
| 22-35秒 | 打开 Word/Pages。按 Fn，说："根据合同法第一百一十四条，当事人可以约定违约金的数额，但不得超过实际损失的百分之三十" → 法律术语全部精准 | "法律术语——违约金、合同法、当事人——全都准。" |
| 35-37秒 | **医学篇开始** — 切到"医学"纠错包 | |
| 37-50秒 | 打开备忘录。按 Fn，说："患者主诉头痛三天，查体血压 150/95，建议口服氨氯地平 5mg qd，复查肾功能" → 医学术语精准 | "医学简写——氨氯地平、qd、肾功能——全都听得懂。" |
| 50-52秒 | **金融篇开始** — 切到"金融"纠错包 | |
| 52-65秒 | 打开文档。按 Fn，说："本季度 EBITDA 同比增长 12%，市盈率从 22 倍降到 18 倍，主要受到汇兑损失的影响" → 金融术语精准 | "EBITDA、市盈率、汇兑损失——再也不会被识别成乱七八糟的东西。" |
| 65-78秒 | 展示自定义热词：手动添加几个公司特有的词 | "还能加你自己的词——公司名、产品名、缩写，随便加。" |
| 78-90秒 | Logo + 标语 | "Xisper。听得懂你的行话。" |

## 注意事项

- 三个行业快速切换，展示广度——不要在某一个上停太久
- 纠错包的开关动作要在画面上清晰可见
- 精准度是这个视频的一切——录之前先排练一遍，确保识别结果没问题
