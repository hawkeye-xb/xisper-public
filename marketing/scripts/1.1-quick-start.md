# 1.1 快速上手 — 下载到首次听写

> Landing Page 主视频。目标：证明"1 分钟上手"是真的。

## 时长：60秒

## 脚本

| 时间 | 画面 | 旁白/字幕 |
|------|------|----------|
| 0-5秒 | xisper.app 官网，鼠标在"Download for Mac"按钮上 | "一分钟，够了。" |
| 5-12秒 | DMG 打开 → 拖 Xisper 到应用程序 → 启动（2倍速） | （无旁白，加速展示拖拽动画） |
| 12-20秒 | 首次启动 → 辅助功能权限弹窗 → 授权 | "授权辅助功能——这样 Xisper 才能帮你打字。" |
| 20-28秒 | 登录页面 → 登录 | "登录一下就好。" |
| 28-35秒 | 菜单栏图标出现。打开 Apple 备忘录，点进一条笔记 | "打开任意 App，点到你想打字的地方。" |
| 35-50秒 | 按住 Fn → 说："今天下午三点跟产品团队开会，讨论一下新版本的发布计划" → 文字实时流式出现 → 松手 → AI 润色后的文本出来 | "按住 Fn，自然地说。文字实时跟着出来。松手——AI 自动润色。" |
| 50-58秒 | 快速切换：在 Slack 里说一句 → 在 Cursor 里说一句 | "任何 App 都能用。Slack、邮件、代码编辑器——有输入框的地方就行。" |
| 58-65秒 | Xisper logo + "免费下载" | "Xisper。按 Fn，开口说。" |

## 注意事项

- 不要展示设置或高级功能——这个视频只讲"多快能用起来"
- 35-50秒是核心高光（实时流式效果），网络一定要稳
- 如果可以，展示浮窗出现的波形动画效果
