# Xisper 运营与推广

## 内容策略

核心原则：**用真实场景展示价值，不拉踩竞品。**

我们和 Typeless、闪电说有很多共通之处——这恰恰说明市场方向是对的。我们的优势在于 **价格、省心、行业精准**。让产品自己说话。

## 录屏前检查清单

- [ ] 桌面干净：隐藏 Dock 无关图标、简洁壁纸、关通知
- [ ] 网络稳定（最好有线）
- [ ] 提前打开要用的 App：Slack、邮件、Cursor、备忘录
- [ ] 开启"开发者"纠错包（开发者演示用）
- [ ] 准备好待编辑的草稿文字（语音编辑演示用）
- [ ] 录屏工具就位：Screen Studio 或 OBS
- [ ] 系统声音关掉（避免 ASR 录到播放的声音）
